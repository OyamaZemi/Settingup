# Mac の初期設定

## 1. このページの目的

このページでは，Mac で開発環境を整える前に必要な共通準備だけをまとめる．  
Python，Git，Julia など個別の設定は，それぞれのページを参照すること．

ここで行うのは，次の 2 つである．

1. Command Line Tools を入れる  
2. Homebrew を入れる


## 2. Command Line Tools

Mac では，開発系のツールを使う前に Command Line Tools を入れておく．
ただし Homebrew をインストールするならその時にまとめてインストールされるので以下は不要。
ターミナルを開いて，次を実行する．

```bash
xcode-select --install
```

すでに入っている場合は，その旨が表示される．



## 3. Homebrew

Homebrew は macOS 用のパッケージマネージャである．
Git などのツールをコマンドで入れたいときに使う．

Homebrew の公式ページにある案内に従ってインストールする．
インストール後，次で確認する．

```bash
brew --version
```



## 4. ターミナルの基本

ターミナルは，コマンドを入力してファイルやツールを操作するためのアプリである．
Launchpad で Terminal を検索するか，`/Applications/Utilities` から開ける．

最初のうちは，次のコマンドだけ分かれば十分である．

```bash
pwd
```

今いるフォルダを表示する．

```bash
ls
```

今いるフォルダの中身を表示する．

```bash
cd フォルダ名
```

指定したフォルダに移動する．

```bash
cd ..
```

一つ上のフォルダに戻る．

```bash
mkdir フォルダ名
```

新しいフォルダを作る．

```bash
clear
```

画面表示を消す．

コマンドを打っていて分からなくなったら，今どこにいるかを `pwd` で確認し，ファイル一覧を `ls` で確認するとよい．
